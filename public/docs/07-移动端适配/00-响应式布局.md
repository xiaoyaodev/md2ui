# 响应式布局

验证移动端适配的各项功能。

## 断点设计

| 屏幕宽度 | 模式 | 布局变化 |
| --- | --- | --- |
| &gt; 768px | 桌面端 | 三栏布局，侧边栏常驻 |
| &lt;= 768px | 移动端 | 单栏布局，抽屉式导航 |

## 移动端特性

### 抽屉式导航

- 侧边栏以抽屉形式从左侧滑出
- 打开时显示半透明遮罩
- 点击遮罩关闭抽屉

### 移动端顶栏

- 独立的移动端 Header 组件
- 包含汉堡菜单按钮、Logo、搜索按钮

### 移动端 TOC

- 底部浮动目录面板
- 点击按钮展开/收起
- 选择标题后自动关闭面板

## 验证要点

1. 将浏览器窗口缩小到 768px 以下，应切换为移动端布局
2. 点击汉堡菜单按钮，侧边栏应以抽屉形式弹出
3. 点击遮罩应关闭抽屉
4. 底部应显示 TOC 浮动按钮
5. 恢复窗口宽度后应自动切回桌面端布局
