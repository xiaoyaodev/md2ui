# 新建与删除

验证侧边栏的文档/目录创建和删除功能。

## 新建文档

1. 点击侧边栏"文档目录"旁的 `+` 按钮
2. 选择"新建文档"或"新建目录"
3. 输入名称，点击确定

### 新建规则

| 规则 | 说明 |
|------|------|
| 名称不能为空 | 提示"名称不能为空" |
| 禁止特殊字符 | `\ / : * ? " < > \|` 不允许 |
| 不能重名 | 同目录下已存在同名文件/目录时提示 |

### 在子目录下新建

- 右键点击文件夹节点的 `+` 按钮
- 新建的文件/目录会创建在该文件夹下

## 删除文档

1. 右键点击文档/目录节点的删除按钮
2. 弹出确认弹窗
3. 确认后删除

### 删除规则

- 删除文件：直接删除
- 删除目录：递归删除目录下所有内容，弹窗会提示"目录下的所有内容将被一并删除"

## 文件热更新

- 文件列表变化后自动刷新导航树（轮询 + ETag 机制）
- 当前文档内容变化后自动刷新内容
- 轮询间隔 1 秒

## 验证要点

1. 点击 `+` 按钮应弹出新建菜单
2. 新建文档后应立即出现在导航树中
3. 删除文档前应弹出确认弹窗
4. 删除后文档应从导航树中消失
5. 外部修改文件后，页面应自动刷新
